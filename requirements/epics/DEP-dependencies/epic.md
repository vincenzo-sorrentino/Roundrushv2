## Objective
Provide visibility into cross-module and cross-epic dependencies so teams can assess impact, plan sequencing, and avoid integration conflicts.

## In scope
- Dependency graph visualisation
- Impact analysis on module changes
- Blocking / blocked status tracking

## Out of scope (for this EP)
- Automated dependency detection from source code
- External third-party dependency management
- Version-level dependency resolution
