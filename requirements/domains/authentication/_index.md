# Authentication Domain

## Scope

Authentication covers account access, organization entry, and identity verification across login, registration, and invitation flows.

## Flow Inventory

- `login`: returning user login via password and controlled failure handling.
- `registration`: self-serve signup and verification path.
- `invitation`: organization invite acceptance and role assignment.
